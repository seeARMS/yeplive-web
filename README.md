#Yeplive Web Client
###Technologies
* bootstrap3
* jquery
* google maps javascript sdk
* gmap3 library

###Prerequisites
* node
* bower

###Installation
1. `bower install`

2. `npm install`

3. `npm start`

Server should now be running on port 3000

###Javascript API
`App` global object

####App.Map
`addMarker(latitude, longitude, title);`

adds a yeplive marker to the map with the given title
